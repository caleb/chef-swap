swap Cookbook
=============
The swap cookbook lets you manage swap files on Linux systems. This is useful to create a swap file, perform some expensive task and then disable it.

Requirements
------------

none - this just uses the built in chef resources


Attributes
----------

none - just use the resources provided


Usage
-----

```ruby
swap_file '/mnt/1GB.swap' do
  size '1gb'
  action :enable
end

compile_some_big_project '...' do
  ...

  notifies :disable, 'swap_file[/mnt/1GB.swap]'
end
```
